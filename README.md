## 📌 Overview

GameSentinel classifies game stream reviews as **Positive** or **Negative** using a Naive Bayes NLP pipeline. It handles gaming-specific slang, noisy text, and informal language with a custom preprocessing step.

## ⚙️ Setup & Run

### 1. Clone the repo
```bash
git clone https://github.com/harshit-69/Game-Stream-Review-Classifier.git
cd Game-Stream-Review-Classifier
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train the model
Make sure your CSV has `review` and `label` columns (Positive / Negative):
```bash
python train.py --data data/reviews.csv
```

### 4. Run the app
```bash
python app.py
```
Open → http://localhost:5000

## 🧠 ML Pipeline

```
Raw Text
   ↓
Punctuation Removal
   ↓
Stopword Filtering (NLTK)
   ↓
CountVectorizer (Bag of Words)
   ↓
TF-IDF Transformer
   ↓
Multinomial Naive Bayes
   ↓
Sentiment: Positive / Negative
```

## 🌐 API

| Endpoint    | Method | Description                  |
|-------------|--------|------------------------------|
| `/`         | GET    | Main UI                      |
| `/predict`  | POST   | JSON `{"review": "..."}` → sentiment result |
| `/health`   | GET    | Server + model status check  |

**Example:**
```bash
curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{"review": "Amazing game, the graphics are stunning!"}'
```

**Response:**
```json
{
  "label": "Positive",
  "emoji": "😄",
  "color": "positive",
  "confidence": 94.2,
  "review": "Amazing game, the graphics are stunning!"
}
```
